Purchase Order Field Extractor

This app extracts explicit fields from purchase order text into a deterministic structured JSON result.

When to use it

Use this app when a workflow has raw purchase order text and needs a read-only extraction of fields such as purchase order number, supplier, buyer, line items, quantities, prices, currency, delivery date, shipping address, and notes.

Accepted input

The MCP tool accepts one field: purchase_order_text. The value must be the raw purchase order text supplied by the user.

Returned output

The app returns a fixed JSON structure with status, extracted fields, item objects, missing fields, the original source text, and structured errors when input is missing, invalid, or out of scope.

What it does not do

This app does not create purchase orders, approve purchase orders, submit orders, validate supplier accounts, check inventory, calculate taxes, generate payments, send emails, contact suppliers, or update external systems.
